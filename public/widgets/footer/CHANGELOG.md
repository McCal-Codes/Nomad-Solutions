# Footer Widget Changelog

All notable changes to the Nomad Solutions Footer Widget will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-26

### Added
- Initial release of footer widget
- Four-column responsive grid layout
- Company information section
- Quick links navigation section (Fleet, Features, Availability, About)
- Contact information section (email, phone, hours)
- Locations section (Reno, Las Vegas)
- Copyright footer with current year
- Desert color palette integration (#6B4C4C, #D4A574, #d1d5db, #a1a1aa)
- Hover effects on links (white color transition)
- Responsive design (stacks to single column on mobile)
- Self-contained HTML + CSS (no JavaScript needed)
- No external dependencies

### Styling
- Dark brown background (#6B4C4C)
- Warm tan headings (#D4A574)
- Light gray text (#d1d5db)
- Darker gray for copyright (#a1a1aa)
- 80px top padding, 24px bottom padding
- 56px gap between columns (40px on mobile)
- Smooth color transitions on link hover (0.2s)

### Navigation Links
All links use anchor format for single-page navigation:
- Fleet → #fleet
- Features → #features  
- Availability → #availability
- About → #about

### Sections
1. **Nomad Solutions** - Company description
2. **Quick Links** - Main navigation shortcuts
3. **Contact** - Email, phone, business hours
4. **Locations** - Service areas

### Browser Support
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+
- Mobile browsers (iOS Safari, Chrome Mobile)

### Integration
- Squarespace-ready (code block compatible)
- Wix-ready (HTML embed compatible)
- WordPress-ready (custom HTML block compatible)
- Works on any HTML page

# Header Widget Changelog

All notable changes to the Nomad Solutions Header Widget will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-26

### Added
- Initial release of header navigation widget
- Sticky header with desert gradient background
- Desktop horizontal navigation menu
- Mobile hamburger menu with slide-down animation
- Optional top contact bar (phone and email)
- Smooth scroll anchor link support
- Desert color palette integration (#6B4C4C, #A85B5B, #D4A574, #F5EFE6)
- Unique function names to prevent Squarespace conflicts (toggleNomadMenu, nomadNavLinks)
- Click-outside-to-close functionality for mobile menu
- Hover effects with warm tan underline animation
- Responsive design for mobile, tablet, and desktop
- Self-contained HTML + CSS + JavaScript (no dependencies)
